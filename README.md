# Cat-Dog-Fox-Duck

React приложение для просмотра случайных фотографий котов, собак и лис.

## Возможности

- Просмотр случайных фотографий котов, собак и лис
- Копирование ссылки на изображение
- Навигация между категориями животных

## Технологии

- React 18
- React Router DOM
- Framer Motion
- React Icons

## Запуск

```bash
npm install
npm start
```

## Сборка

```bash
npm run build
```
